#  Hotel Booking System - MySQL

A complete relational database project built using MySQL.

##  Tables
- Hotels
- Rooms
- Room Types
- Guests
- Bookings
- Payments
- Staff
- Services

##  Tools Used
- MySQL
- MySQL Workbench

##  Queries Included
- Revenue per hotel
- Most booked room types
- Guest booking history
- Upcoming bookings
- Average review ratings

##  How to Run
1. Open MySQL Workbench
2. Open hotel_booking_system.sql
3. Press Ctrl + Shift + Enter to run
